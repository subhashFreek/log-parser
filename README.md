# Log Parser


## Backend

#### Requirements

- Node.js v14+
- TypeScript

## Installation

 Navigate to the project directory
    ```
    cd log-parser/backend
    ```
run ``` npm i``` and then ```npm run build``` and then ```npm start```

## Frontend

#### Requirements

- React 14.0.0+

## Installation

 Navigate to the project directory
    ```
    cd log-parser/frontend
    ```
run ``` npm i``` and then ```npm start```
